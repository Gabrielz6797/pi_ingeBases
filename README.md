# pi_ingeBases
Repository for course CI-0128: "Proyecto Integrador de Ingeniería de Software y Bases de Datos", group: Ta' Bueno.

## Meaning of "UserType" values:
0: Active employer.
1: Active employee.
2: Inactive employer.
3: Inactive employee.

## Meaning of "ContractType" values:
0: Full-time Employee
1: Half-time employee
2: Hourly employee
3: Professional services
